DbgScript

extends WinDbg (and cdb) with scriptability in several popular programming languages.

Elegant API

Care was taken in the design of the object model for interaction with the debugger.

Use what you know

DbgScript comes with 3 popular languages right out of the box: Ruby, Python and Lua!

Feature and API parity a reality and goal

Feature parity across all first-party languages is a reality and remains a goal.

DbgScript is free and open

DbgScript is released under the MIT license, so you can use it for whatever you want. Contributions welcome too!
